Escritor Pro

Aplicación web para móviles · Dictado de voz

Aplicación web pensada para móviles que permite crear un diario de notas por dictado de voz de forma simple y eficaz. Convierte voz en texto, organiza el contenido por categorías y permite su edición y exportación, manteniendo todos los datos almacenados localmente para garantizar la privacidad.
